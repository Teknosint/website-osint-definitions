The OSINT Definitions Project

The OSINT Definitions Project brings together a comprehensive collection of definitions for Open Source Intelligence (OSINT). With so many differing interpretations across academia, OSINT practitioners, national/international organizations, and more, this project aims to make it easier to explore and compare every definition in one place. This project is designed to support researchers, professionals, and anyone intrigued by the lack of commonly agreed-upon definition.

If you have any questions, feedback, or are interested in collaborating, feel free to reach out at teknosint@gmail.com! Also, please let me know if you find any definition that is not included here.
